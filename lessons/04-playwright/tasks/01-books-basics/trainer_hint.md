## Trainer Notes

- Make sure participants can run the test suite locally before starting this task.
- Encourage them to use **role- or test‑id based locators**, not brittle CSS selectors.
- If time allows, nudge them towards extracting a simple `BooksPage` POM so later tasks can reuse it.
- Watch out for unnecessary `waitForTimeout` / `sleep` calls and point them to Playwright's auto‑waiting instead.
